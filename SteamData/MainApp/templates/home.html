This Project:

This is an interdiscplinary capstone between Computer Science (CS), Management Information Systems (MIS), and Web Development (WEB). The goal of this project was to create a relational database and web application on the records of video games on the Steam digital distribution service. By blending these disciplines, this project proves that there is merit in comprehensive and interdisciplinary knoweledge.

The PC games included in this database, are the 129 most popular games in June and July of 2025. Over the course of six weeks, the app ID, title, rank, and player count of these games were manually collected in order to avoid webscraping the Steam website. The app ID was then used to call an API that would return a JSON object with more data about the game. After the retrieval, all the data was cleaned with the Pandas libarary, coalesced into a single line of a CSV, and normalized into Boyce-Codd normal form and formatted for My SQL Server.

This web application was built with the Django framework and deployed with Railway. It aims to present the project and serve as documentation. This website is accessible and responsive. The design of the website was created custom CSS and the Bootstrap library.
